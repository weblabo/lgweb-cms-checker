# 自治体サイト CMS Checker

閲覧中の自治体サイトが、どのCMS（コンテンツ管理システム）で構築されているかをワンクリックで判定するChrome拡張機能です。さらに、サイトの「リニューアル日」も同時に表示します。
自治体サイト以外でも動作します。
CMS、リニューアル日は間違っている場合があるので、あくまでも参考情報としてください。

## 主な機能

- **ワンクリックCMS判定**: 現在開いているタブのHTMLソースを解析し、使用されているCMSを瞬時に判定してポップアップに表示します。
- **リニューアル日の自動取得**: 表示中サイトのサイトのリニューアル日を表示します。
- **軽量・高速**: 外部APIに依存せず、ブラウザ内とローカルファイルのみで完結するため、高速に動作します。

## インストール方法（開発者モード）

Chromeのウェブストアを経由せず、ローカルから直接インストールして使用します。

1. このリポジトリをダウンロード（または `git clone`）し、任意のフォルダに展開します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力して拡張機能の管理画面を開きます。
3. 画面右上の **「デベロッパー モード」** をオン（有効）にします。
4. 画面左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5. 手順1で展開したフォルダ（`manifest.json` が含まれているフォルダ）を選択します。
6. Chromeのツールバーに本拡張機能のアイコンが追加されれば準備完了です！

## 使い方

1. 調べたい自治体などのWebサイトをブラウザで開きます。
2. ツールバーの拡張機能アイコンをクリックします。
3. ポップアップが開き、現在のURL、判定されたCMS名、およびリニューアル日が表示されます。
