# 自治体 CMS Checker

閲覧中の自治体サイトが、どのCMS（コンテンツマネジメントシステム）で構築されているかを判定するChrome拡張機能です。独自のデータベースに基づき、サイトの「リニューアル日」も同時に表示します。

自治体公式サイトの分析や、アクセシビリティ対応の調査をサポートします。
※CMS名およびリニューアル日は解析結果に基づく推定値であるため、あくまで参考情報としてご利用ください。

## 📥 インストール

Chromeウェブストアからインストールいただけます。
[Chromeウェブストア - 自治体サイト CMS Checker](https://chrome.google.com/webstore/detail/nmkglebleknfeommicflfhkmojlaodpp)

## ✨ 主な機能

- **高精度なCMS判定**: 開いているタブのソースコードをスキャンし、独自の判定ロジックにより使用CMSを特定します。
- **リニューアル日の表示**: 自治体ドメインデータベースと照合し、判明しているリニューアル日をポップアップ内に表示します。
- **リニューアル予定の通知**: 未来の日付が登録されている場合、「リニューアル予定」として強調表示されます。
- **ステータス連動アイコン**: ポップアップを開いている間、ツールバーのアイコンがアクティブ状態に変化し、実行中であることを視覚的に伝えます。
- **アクセシビリティ対応デザイン**: WCAG 2.1を意識したコントラスト比を採用し、視認性の高いUIを実現しています。

## 🛠 開発者向け：ローカルでのインストール手順

開発用ソースコードから直接インストールする場合は、以下の手順に従ってください。

1. このリポジトリをダウンロード（または `git clone`）し、任意のフォルダに展開します。
2. Google Chromeのアドレスバーに `chrome://extensions/` と入力して管理画面を開きます。
3. 画面右上の **「デベロッパー モード」** をオンにします。
4. **「パッケージ化されていない拡張機能を読み込む」** をクリックし、展開したフォルダを選択します。

## 📝 免責事項

本拡張機能は、公開されているHTML情報および独自に収集したデータに基づいて判定を行っています。ウェブサイトの構成変更等により、正しく判定されない場合があります。
また、CMSによっては誤判定を行う場合があります。
